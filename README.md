# guide-mkdefi

本仓库用于归档和发布多个独立 HTML 页面。这些页面本身不依赖特定域名或网站，仅作为静态资源的集合与发布入口。

## 项目简介

`guide-mkdefi` 是一个静态页面归档仓库，主要存放各类独立的 HTML 文档。每个页面均可独立访问，适合用于信息展示、工具页面、演示文档等场景。仓库结构简单，便于快速浏览和引用。

## 目录说明

```
guide-mkdefi/
├── pages/        # 存放所有独立 HTML 页面
├── assets/       # 共用资源文件（如样式、脚本、图片等）
└── README.md     # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名通常反映其内容或用途。
- `assets/`：存放页面可能依赖的静态资源，按需组织子目录。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开。
- 页面间不存在强制跳转或依赖关系，可单独使用或部署。
- 仓库不维护任何在线服务，仅提供文件存储与版本管理。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在 `pages/` 目录下找到感兴趣的 HTML 文件。
3. 双击或在浏览器中直接打开该文件即可查看内容。

如需部署到 Web 服务器，请将仓库根目录作为网站根目录，或按需配置路径。
